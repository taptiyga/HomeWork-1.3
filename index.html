Написать функцию, которая принимает два числа, первое и второе значение.

  • С помощью цикла, необходимо организовать алгоритм, который должен выполняться столько раз, чему равно второе значение.
  • И необходимо первое значение(первый параметр) умножить на себя столько раз, чему равно второе значение.
  • Но при этом задание необходимо выполнить так, чтобы умножение происходило без умножения, т.е путём сложения первого числа на себя.

Примечание:

  • 1) Добавь проверку входных данных (оба параметра должны быть числами, второе значение не может быть отрицательным).
  • 2) Запрет использования встроенных методов массивов или других упрощающих решений
function Exponentiation(a, b) {
   if (typeof a === 'number' && typeof b === 'number' && b >= 0) {
      let sum = 1
      for (let index = 0; index < b; index++) {
         sum = sum * a
      }
      console.log(sum)
   } else {
      console.log(`error:оба параметра должны быть числами, второе значение не может быть отрицательным`)
   }
}
Exponentiation(3, 10)
